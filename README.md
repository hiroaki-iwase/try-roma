# Try ROMA
virtual console of ROMA

## Free mode
You can try to use below ROMA commands.
### GET command
- stat
- stats
- whoami
- nodelist
- get
- gets

### DELETE command
- balse
- shutdown
- shutdown_self
- (rbalse)

### POST command
- set
- add
- delete
- replace
- append
- prepend
- cas
- set_expt
- incr
- decr

### PUT command
- release
- recover
- set_auto_recover
- set_log_level

Usage of each commands, please refer to the [ROMA HP](http://roma-kvs.org/commands.html).

## Tutorial Mode
This mode explain the ROMA basic commands as a interactive.
